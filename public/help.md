# Guía de BroWorks Dev-Caddy

Bienvenido a tu nueva estación de comando.

## 🚀 Conceptos Básicos

### 1. Categorías y Comandos
Navega por las categorías en el panel izquierdo. Haz clic en una tarjeta para **copiar** el comando al portapapeles.

### 2. Variables Dinámicas
Algunos comandos tienen variables como `{branch}` o `{ip}`.
- Rellena los campos que aparecen en la tarjeta.
- El comando se actualizará automáticamente.
- Al copiar, obtendrás el comando final listo para usar.

### 3. Modo Zen
La interfaz está diseñada para evitar distracciones. Todo ocurre en una sola página.

## 🛡️ Modo Edición (Admin)

Activa el interruptor **Desbloquear/Bloquear** en la barra lateral para entrar en **Modo Edición**.

### Funciones Admin:
- **Crear**: Usa el botón flotante `+` para añadir nuevos comandos.
- **Reordenar (Drag & Drop)**: Arrastra categorías o comandos para organizarlos a tu gusto.
- **Editar/Eliminar**: Usa el menú `...` en cualquier tarjeta o categoría.

### Gestión de Datos
- **Favoritos**: Haz clic en la estrella ⭐ para guardar comandos frecuentes.
- **Backup**: Usa el botón de "Descargar / Subir" en la barra lateral para exportar o importar tu configuración en JSON.
